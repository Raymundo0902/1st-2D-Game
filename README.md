⚠️ **Work In Progress**

Pinewood Camp Massacre is a 2D survival horror game built in **Java** where the player must navigate a forest environment while being hunted by an unwelcomed entity.

This project focuses on building core game development systems from scratch, including entity architecture, tile-based worlds, collision detection, and enemy AI using the **A*** **pathfinding algorithm**.

The goal of this project is to explore game systems, artificial intelligence behavior, and real-time player interaction.

## Current Features

- **A*** **Pathfinding AI**
  - Enemies dynamically navigate the map to chase the player.
  - Uses node-based pathfinding with G, H, and F cost calculations.

-  **Tile-Based World System**
   - Game maps are built using tile grids.
   - Supports collision tiles and environment interaction.

-  **Entity System**
   - Modular architecture for players, monsters, and objects.

-  **Collision Detection**
   - Prevents entities from walking through walls and    obstacles.

-  **Interactive environment Objects**
   - Objects such as campfires affect player movement and pathfinding.

## Project Structure

    main/:
    Game loop and core engine logic 

    ai/:
    Where pathfinding algorithm is implemented

    entity/:
    Monsters, NPC's, Player, etc

    object/:
    Interactive objects in world

    tile/:
    Handles map loading
